# 📖 덤프링(Dumpring) 용어 정의 및 변경 검토표

본 문서는 덤프링 프로젝트 소스 코드 및 설계에 사용된 비즈니스 용어, 데이터베이스 테이블명, 상태 코드 등을 파악하여 새로운 명칭으로 재정의하기 위한 검토 문서입니다. 
`바꿀 이름` 열의 빈칸을 채워 시스템 표준화에 반영할 수 있습니다.

---

## 1. 🏗️ 비즈니스 도메인 및 역할 용어

| 현재명 (시스템 명칭) | 바꿀 이름 (변경안 기재) | 용어 설명 (Description) |
| :--- | :--- | :--- |
| **상차지 (Loading Site)** | | 흙을 싣는 출발 공사현장. 굴착기가 토사를 덤프 트럭에 적재하는 장소 |
| **하차지 (Unloading Site / Drop-Off)** | | 흙을 내려놓는 도착 사토장/매립지. 지주(땅주인)가 소유하고 운영 |
| **사토장 (Drop-Off Site)** | | 건설 현장에서 발생한 잔토/토사를 합법적으로 성토 및 매립하는 부지 |
| **토사 / 사토 (Material / Soil)** | | 건설 현장에서 굴착 및 발생하여 운송되는 흙, 진흙, 돌 등의 총칭 |
| **토종 (Material Type)** | | 토사의 세부 성상 종류 (양질토, 뻘흙, 암버럭, 혼합토 등) |
| **양질토 (GOOD_SOIL)** | | 성토 및 도로 공사용으로 매우 우수하고 규격에 부합하는 깨끗한 흙 |
| **뻘흙 (MUD_SOIL)** | | 점성이 강하고 수분 함량이 높아 반입 시 정밀 검사가 필요한 토사 |
| **암버럭 (ROCK)** | | 발파, 터널 굴착 시 발생하는 돌덩어리 및 암석류 화물 |
| **혼합토 (MIXED)** | | 양질의 흙과 암석 또는 기타 토사가 혼재된 복합 토사 |
| **오더 (Order / JobPost)** | | 현장관리자가 기사를 수배하기 위해 등록하는 배차 요청 공고 |
| **배차 (Dispatch)** | | 특정 오더(모집공고)에 기사 및 차량을 배정하여 운행을 지정하는 행위 |
| **배차 티켓 (Dispatch Ticket)** | | 기사 1명의 1회 운행 주행을 실시간 트래킹하고 정산하는 최소 단위 |
| **콜 (Call)** | | 기사 모바일 화면에 제공되는 배차 요청 알림 수신 정보 |
| **미터기 (Fare Meter)** | | 실시간 GPS 위치 및 속도를 병산하여 운임을 산정하는 덤프용 미터기 시스템 |
| **지오펜싱 (Geofencing)** | | GPS 가상 경계. 하차지 중심 반경 200m 진입 시 자동 도착 감지 트리거 |
| **회차 (Return / Rejection)** | | 하차지 토질 검사 부적합 등으로 반입 거부되어 상차지로 돌아가는 상황 |
| **세륜기 (Washing Facility)** | | 현장 및 사토장 입출차 시 트럭 바퀴의 진흙을 세척하는 필수 설비 |
| **선등록 (Pre-registration)** | | 소장/차주가 번호만으로 소속 기사/직원을 사전에 연동 대기 등록하는 구조 |
| **매칭 (Matching)** | | 상차지 현장과 하차지 사토장을 상호 연결하는 B2B 계약 중개 관계 |
| **수용 공고 (Drop-Off Request)** | | 하차지 지주가 흙을 받기 위해 등록하는 매립 수용 조건 공고 |
| **현장 관리자 (Site Manager)** | | 공사현장을 개설하고 상차 배차 오더를 생성 및 결제하는 책임자 (소장/경리) |
| **현장 담당자 (Site Worker / Staff)** | | 상차지 게이트에 상주하며 덤프 트럭의 진출입을 실시간 승인하는 직원 |
| **차주 (Owner)** | | 덤프 트럭을 소유하고 소속 기사 및 운송 정산을 총괄 관리하는 사장님 |
| **기사 (Driver)** | | 실제 덤프 트럭을 운행하며 배차를 수행하고 실시간 미터기를 작동하는 운전기사 |
| **하차지 지주 (Drop-Off Owner)** | | 사토장 땅을 소유하고 반입되는 토질을 확인 및 최종 반입 승인하는 주체 |
| **플랫폼 관리자 (Admin)** | | 회원 서류 심사, 단가 및 수수료 관리, 분쟁 발생 시 중재 및 직권 정산 총괄 |

## 2. 🗄️ 데이터베이스 테이블 및 모델명

| 현재명 (시스템 명칭) | 바꿀 이름 (변경안 기재) | 용어 설명 (Description) |
| :--- | :--- | :--- |
| **users (User Table)** | | 다중 역할 토글 필드 및 휴대폰 번호, 본인인증 CI가 저장되는 통합 회원 테이블 |
| **construction_sites** | | 건설사 정보, 현장 좌표, 지오펜스 반경이 정의된 공사현장 마스터 테이블 |
| **site_employees** | | 현장 관리자가 소속 직원 관리를 위해 번호로 선등록해 두는 직원 테이블 |
| **unloading_sites** | | 지주가 등록한 사토장 기본 정보 및 preferred_soil_types 수용 정보 |
| **cars (Car)** | | 덤프 차량번호, 톤수(T_15/T_25/T_27) 및 소유 차주가 연결된 등록 차량 테이블 |
| **drivers (Driver Profile)** | | 차주가 선등록한 기사 정보 및 배정 차량 정보가 맵핑되는 기사 프로필 |
| **user_uploaded_documents** | | 회원가입 승인 심사를 위해 업로드한 증빙 서류 파일 보관 테이블 |
| **orders (Order - Legacy)** | | 상차지-하차지 중계 및 기사 모집을 관리하던 구버전 오더 테이블 |
| **site_profiles** | | 현장 관리자/담당자 가입 시 기입되는 소속 건설사 및 사업자 세부 정보 |
| **drop_off_profiles** | | 하차지 지주 가입 시 기입되는 인허가 정보 및 사토장 주소 세부 프로필 |
| **site_user_mappings** | | 현장담당자가 현장에 매핑 신청 시 소장의 승인을 관리하는 다대다 연결 테이블 |
| **common_codes** | | 토사종류, 트럭규격, 정산방식 등 시스템 동적 비즈니스 코드를 보관하는 마스터 테이블 |
| **drop_offs** | | 지주가 개설한 하차지의 좌표(위경도), 주소, 허가번호를 통합한 사토지 마스터 테이블 |
| **drop_off_requests** | | 하차지 지주가 흙을 받겠다고 등록하는 목표 Tonnage 수용 조건 테이블 |
| **job_posts (JobPost)** | | 상하차지 B2B 양방향 매칭 및 기사 모집 배차를 위한 통합 현장 오더 테이블 |
| **driver_favorite_regions** | | 기사가 배차를 받기 위해 사전에 등록해 두는 선호 시도/시군구 구역 테이블 |
| **dispatch_tickets** | | 기사의 개별 1회 운송 주행을 기록하고 실시간 요금을 정산하는 운행 티켓 테이블 |
| **sdui_themes** | | 앱의 전체 UI 테마(배경, 색상, 폰트)를 서버에서 실시간 동적 제어하기 위한 테이블 |
| **wallets** | | 사용자(기사, 차주, 지주)별 실시간 운임 가상 포인트 잔고를 관리하는 가상 지갑 테이블 |
| **wallet_transactions** | | 정산 이체, 출금, 포인트 환전 등의 가상 지갑 트랜잭션 기록 원장 |
| **gps_tracks** | | 운임 계산의 원천이 되는 운행 티켓별 5초 주기의 실시간 위경도 및 속도 기록 테이블 |

## 3. 🔄 상태 및 타입 코드

| 현재명 (시스템 명칭) | 바꿀 이름 (변경안 기재) | 용어 설명 (Description) |
| :--- | :--- | :--- |
| **WAITING_MATCH** | | B2B 오더 중 상차지가 먼저 등록되어 아직 하차지가 정해지지 않은 매칭 대기 상태 |
| **WAITING_APPROVAL** | | 하차지가 매칭 신청을 하였으나 상차지 소장의 최종 승인을 기다리는 상태 |
| **OPEN** | | 매칭이 확정되어 기사들 모바일 앱에 배차 모집 공고로 활성화 노출된 상태 |
| **COMPLETED** | | 덤프 트럭 이송 완료 및 하차지 승인이 완료되어 운송이 최종 종결된 상태 |
| **CANCELLED** | | 현장 사정 또는 매칭 반려 등으로 오더가 철회되거나 폐기된 상태 |
| **ACCEPTED** | | 배차 공고를 기사가 확인하고 수락을 완료한 주행 전 상태 |
| **DRIVING** | | 상차지 출발 버튼 클릭 후 GPS 미터기가 작동하기 시작한 실시간 주행 상태 |
| **ARRIVED (ARRIVED_DROPZONE)** | | 하차지 지오펜싱(200m) 영역 진입이 감지되어 도착 대기 중인 상태 |
| **APPROVED** | | 하차지 지주가 토질 검사 후 정상 반입을 확인하여 정산이 자동 확정 완료된 상태 |
| **REJECTED** | | 토질 부적합으로 인해 반입이 거부(회차)되어 주행 완료가 보류된 상태 |
| **DISPUTED** | | 반입 거부 또는 운임 오차로 인해 기사/소장이 중재 조정을 접수한 분쟁 상태 |
| **SETTLE_ADJUSTED** | | 분쟁 발생 후 어드민이 GPS 경로 분석을 거쳐 70% 직권 정산 등을 강제 실행한 상태 |
| **MATERIAL_TYPE** | | 공통 코드 - 토사 종류 대분류 그룹 코드 |
| **TRUCK_TYPE** | | 공통 코드 - 덤프 트럭 톤수 대분류 그룹 코드 |
| **PAYER_TYPE** | | 공통 코드 - 운송비 지불 주체 분류 그룹 코드 |
| **PAYMENT_METHOD** | | 공통 코드 - 정산 결제 방식 분류 그룹 코드 |
